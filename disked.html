so, what is disked you may ask?

disked is a nas manager / interface for linux with support for 4 disks.

it is a hobby project im making for flavortown

currently, it is in a very early state and buggy

you can get it from github and build it with cmake.

it is written in c++

dependencies if you actually want to build it:

  • gcc or your favourite c compiler
  • cmake
  • a linux machine
